# BookNest
Web site (without styles) on ASP.NET Core (Web API), Entity Framework Core and React for booking rooms in hotel.

## Requirements
- [.NET 8.0 SDK](https://dotnet.microsoft.com/en-us/download/dotnet/8.0)
- [SQL Server](https://www.microsoft.com/uk-ua/sql-server/sql-server-downloads)
- Git

## Installation
Clone repository
```bash
git clone https://github.com/Nikita-Titarenko/BookNest
cd BookNest
```

Set database in appsettings.json
```bash
dotnet ef database update
```

Launch application
```bash
dotnet run
```
